# Apa itu Javascript?
<p>📌 Kalau kita akses website, kita sebenarnya sedang melakukan proses <b>HTTP request</b> dimana kita meminta kepada server, dan akan mendapatkan <b>HTTP response</b> berupa 3 file: HTML, CSS dan Javascript.</p>
<p>Javascript mampu melakukan banyak hal:</p>

* Mengubah HTML dan CSS secara live
* Manipulasi data (karena ia adalah bahasa pemrograman, bukan bahasa markup)
* Respon ke event (jika tombol di klik, scroll, dll)
* Animasi
